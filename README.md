# Test Cases

## Опис завдання

Створення тест-кейсів для перевірки функціоналу сайту.
У рамках роботи було розроблено 3 тест-кейси, що перевіряють:

* коректність відображення перекладу
* відображення детальної інформації подій
* поведінку сайту, при відсутності інтернет-з'єднання
