# Test Cases

## Опис завдання

Створення тест-кейсів та автоматизованих тестів для перевірки функціоналу сторінки подій сайту GreenCity.

У рамках роботи було розроблено 3 тест-кейси та реалізовано відповідні автотести, що перевіряють:

* коректність перекладу фільтрів після зміни мови
* відображення детальної інформації подій після натискання кнопки "Більше"
* поведінку сайту при відсутності інтернет-з'єднання

Автотести реалізовані з використанням Selenium та Python (unittest).

## Посилання на сайт

https://www.greencity.cx.ua/#/greenCity/events

## Інструкція запуску

1. Встановити залежності:

```
pip install -r requirements.txt
```

2. Запустити тести:

```
python -m unittest discover tests
```
